# Daily Check-in Template
*Anchor point for the day — free-form conversation, ~5 minutes. The Diarist runs throughout the day; this is the structured moment.*

---

## Opening

Start with where the user actually is, not where they should be:

- "How are you feeling this morning?" / "How's the day going so far?"
- "Energy level — high, medium, low?"
- "Anything unexpected come up since yesterday?"

## Extract (from conversation, not as a form)

The system should naturally surface these through dialogue — not present them as fields to fill:

- **Mood:** overall emotional tone
- **Energy:** physical and mental capacity today
- **Focus:** what feels easiest to engage with right now
- **Blockers:** anything in the way
- **Wins:** anything worth noting from yesterday

## Direction

Hand off to the Time Director for the day's plan once context is established.

## Close

- Confirm the log has been written
- Note anything the Diarist should follow up on later

---

*This template evolves. Edit it directly or through the tool as patterns emerge.*
