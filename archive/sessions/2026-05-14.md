# Session Log — 2026-05-14 — Kokoro TTS Setup & Voice Selection

## What Was Done

### TTS: Replaced Piper with Kokoro

- Piper (lessac-high) was evaluated and rejected as "wrong feel, not lifelike"
- edge-tts (Microsoft neural, JennyNeural) was tried as an intermediate step
- Kokoro 0.9.4 selected as primary TTS — local, no API key, no limits

**Installation:**
- `brew install python@3.12` — Kokoro requires 3.10–3.12; project venv is 3.14
- `brew install espeak` then `brew unlink espeak && brew install espeak-ng`
- Kokoro venv isolated at `tools/kokoro/venv/` (Python 3.12, separate from project .venv)
- Setup script: `tools/kokoro/setup.sh`
- Speaker script: `tools/kokoro/speak.py`
- Audition script: `tools/kokoro/audition.py`

**TTS fallback chain in `core/voice_pipeline.py`:**
Kokoro → edge-tts → Piper (in order of preference)

### Voice Selection — Two Blind Audition Rounds

**Round 1 shortlist** (9 voices, randomized):
- Liked: af_heart, af_bella, af_sky, am_puck, am_liam, am_michael, bf_isabella, bf_lily, bm_lewis
- Dismissed: all others from full voice list

**Round 1 preference order:**
- Male: am_liam > am_puck > bm_lewis > am_michael
- Female: shortlisted — af_heart, af_sky, bf_isabella, bf_lily (af_bella dismissed)
- Decision: female voice chosen for the project

**Round 2** (4 female voices, randomized blind):
- Preference order: af_heart > bf_lily > bf_isabella > af_sky

**Active voice: `af_heart`** (set in `core/voice_pipeline.py` → `KOKORO_VOICE`)
