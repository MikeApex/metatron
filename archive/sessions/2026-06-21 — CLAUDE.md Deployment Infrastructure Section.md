## What the new section covers

1. **Topology diagram** — Mac → GitHub → VM → Vertex AI; phone → Tailscale → VM

4. **Billing protection** — $20 hard cap, Pub/Sub topic `billing-cap`, Cloud Function `stop-billing`, retry policy
5. **Tailscale** — device IPs, setup command, why HTTP (not HTTPS) is acceptable
6. **systemd services** — exact unit file contents for `metatron-server.service` and `metatron-scheduler.service`, management commands
7. **GitHub + deploy pipeline** — account, repo, SSH keys (Mac vs VM), `deploy.sh` behavior, post-commit hook
8. **Python environment** — venv setup, why Kokoro has a separate venv
9. **Environment variables** — every variable with source and purpose
10. **Routing / deployment mode** — `DEPLOYMENT_MODE` values, which yaml loads for each
11. **Android app** — Capacitor config, server address constant, icon, build prerequisites, full build + sideload steps, when to rebuild vs. when not to
12. **Local dev mode** — Ollama, sleep prevention, TTS voice, Whisper model, TLS certs
13. **Recreate from scratch** — 10-step ordered checklist: GCP project → billing cap → service account → VM → Tailscale → GitHub → repo transfer → systemd → deploy pipeline → APK
